# 50_00_VIGIL — Índice del Módulo VIGIL

> **UDPT-NC** | Fase 50 | Origen: `Agency/vigil/`

## Descripción

VIGIL es el subproyecto de vigilancia y análisis de inteligencia electoral.
Contiene su propia arquitectura interna con separación src/data/notebooks.

## Estructura Interna

| Directorio | Descripción |
|-----------|-------------|
| `src/` | Código fuente: ingestion, processing, analysis, graph |
| `data/` | Datos en niveles: raw → silver → gold |
| `config/` | Templates de configuración |
| `notebooks/` | Notebooks de análisis VIGIL |
| `evals/` | Evaluaciones y benchmarks |
| `docs/` | Referencias y documentación interna |
| `reports/` | Reportes generados |
| `bin/` | Scripts ejecutables |

## Nota de Integración

VIGIL es un submódulo autónomo. Sus outputs finales deben volcarse a:
`Agency/80_00_OUTPUTS/80_10_REPORTS/`
