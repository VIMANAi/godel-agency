## 📐 VARIABLES PARA MODELO PDIV

### Sentimiento Esperado vs Real

Basado en intención de voto Rubrum, el **sentimiento digital** debería correlacionar:

```
PDIV Esperado = (Intención_Voto × 0.6) + (Engagement_Rate_Normalizado × 0.4)

Donde:
- Intención_Voto: Normalizada 0-100
- Engagement_Rate_Normalizado: (ER_candidato / ER_maximo) × 100
```

**Desviaciones a investigar:**
- Si PDIV > Intención_Voto: Candidato con **momentum digital** (oponentes subestiman)
- Si PDIV < Intención_Voto: Candidato con **voto duro tradicional** (riesgo de baja participación)
